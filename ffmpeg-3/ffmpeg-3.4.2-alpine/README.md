# ffmpeg 3.4.2 alpine 3.7
## Running
```
docker run --rm sitkevij/ffmpeg:3.4.2-alpine
```
## Pulling from Docker Hub
```
docker pull sitkevij/ffmpeg:3.4.2-alpine
```
## Building from source
```
git clone https://github.com/sitkevij/ffmpeg.git && \
cd ffmpeg && \
chmod a+x build-local.sh && \
./build-local.sh sitkevij ffmpeg-3.4.2-alpine && \
docker run --rm sitkevij/ffmpeg:3.4.2-alpine
```
